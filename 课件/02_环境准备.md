# 第 02 课：环境准备 —— 安装 gRPC 全家桶

> **目标**：20 分钟内装好所有必要的工具，为后续实战做好准备  
> **前提**：Python 3.8+ 已安装，pip 可用

---

## 一、核心依赖安装

打开终端，执行：

```bash
pip install grpcio grpcio-tools
```

| 包名 | 干什么用的 |
|------|-----------|
| `grpcio` | gRPC 核心库（Channel、Server、Stub、流式等一切） |
| `grpcio-tools` | 包含 `protoc` 编译器和 Python 代码生成插件 |

### 验证安装

```bash
python -c "import grpc; print(grpc.__version__)"

python -m grpc_tools.protoc --version
# 输出类似: libprotoc 3.xx.x
```

两条命令都无报错则安装成功 ✅

---

## 二、对比学习用依赖

因为我们全程用 FastAPI 做对照，也需要安装 HTTP 相关库：

```bash
pip install fastapi uvicorn httpx
```

| 包名 | 干什么用的 |
|------|-----------|
| `fastapi` | REST 框架（第 06 课对比用） |
| `uvicorn` | ASGI 服务器（启动 FastAPI） |
| `httpx` | HTTP 客户端（调用 FastAPI 服务端） |

### 验证安装

```bash
python -c "import fastapi; print(fastapi.__version__)"
python -c "import uvicorn; print('uvicorn OK')"
python -c "import httpx; print(httpx.__version__)"
```

## 三、可选工具（后面课才用到，先不急着装）

| 工具 | 用途 | 安装方式 |
|------|------|----------|
| `grpcurl` | 命令行调试 gRPC（像 curl） | 需要 Go 环境 |
| `grpcui` | Web UI 调试 gRPC（像 Postman） | 需要 Go 环境 |
| `ghz` | gRPC 压力测试工具 | 需要 Go 环境 |
| `betterproto` | 更 Pythonic 的 proto 生成 | `pip install betterproto` |

> 这些工具在第 12 课才会用到，暂时不装没关系。

## 四、确认安装清单

如果以下命令都能成功执行，环境就准备好了：

```bash
# ✅ 核心 gRPC
python -c "import grpc; print('grpcio:', grpc.__version__)"

# ✅ Proto 编译器
python -m grpc_tools.protoc --version

# ✅ FastAPI 对照组
python -c "import fastapi; import uvicorn; import httpx; print('REST 对照组 OK')"
```
