# 🎓 RPC & gRPC 从入门到实战 — 课程目录

> **适用人群**：有 REST (FastAPI / Spring Boot) 开发经验，想用 Python 系统学习 RPC 和 gRPC

> **授课老师**：AI 编程助手

## 💡 使用说明

1. **按顺序学习**：每课都依赖前面的知识，不要跳课
2. **代码都要跑**：每课的代码都能直接运行，不要只看不写
3. **先看懂再对比**：每 implement 一个 gRPC 功能，马上去看对应的 FastAPI 版本（第 06 课有完整对比）
4. **做笔记**：每课学完后，在对比表上加上你的理解
5. **提问**：遇到不懂的随时问我
